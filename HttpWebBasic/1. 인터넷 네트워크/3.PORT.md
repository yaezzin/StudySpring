# # 1-3. PORT

## Port 란?

* 같은 IP 내에서 프로세스를 구분하는 것
* IP는 목적지 서버를 찾는 것, 서버 안에서 돌아가는 애플리케이션을 구분하는 것을 포트라 생각하자!

## 포트 번호
* 0 ~ 65535 : 할당 가능
* 0 ~ 1023: 잘 알려진 포트, 사용하지 않는 것이 좋음
* ```FTP``` - 20, 21
* ```TELNET``` - 23
* ``` HTTP``` - 80
*  ```HTTPS``` - 443
